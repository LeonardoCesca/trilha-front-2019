HTML (Hypertext Markup Language), é uma linguagem de marcação de Hipertexto considerada a base da internet. Criada por Tim Berners-Lee com o intuito de comunicar e disseminar pesquisas entre ele e seu grupo de colegas, o que na época, começou a ser utilizada para formar a rede pública daquela época. Por ser uma linguagem baseada em marcação, é necessário marcar os elementos para mostrar quais informações a página exibe. Além disso, seu objetivo é ser de fácil entendimento por seres humanos e por máquinas.

CSS (Cascading Style Sheets), é uma linguagem utilizada para definir(setar) a aparência de documentos feitos em HTML. Antes do surgimento do CSS, o HTML exercia essa função, porém, isso ao longo do tempo trouxe problemas e, a partir, desses problemas nasceu o CSS. O seu objetivo é separar o conteúdo de marcação do estilo (aparência) do site. Sua manipulação pode ser feita tanto em arquivo externo quanto dentro do arquivo HTML.

JS (JavaScript) foi criada em 1995 por Brendan Eich, um engenheiro da Netscape, e lançada pela primeira vez com o Netscape 2 no início de 1996. Considerada uma linguagem dinâmica orientada a objetos, possui tipos e operadores, objetos e métodos. Vale a pena ressaltar que o JavaScript não tem classes, em vez disso, a funcionalidade de classe é realizada por protótipos de objetos que dão as funções a capacidade para armazenar código executável e serem passadas como parametro para qualquer outro objeto.

ASCII é uma sigla para “American Standard Code for Information Interchange” (Código Padrão Norte-americado para Intercâmbio de Informações). Esse código foi proposto por Robert W. Bemer, tendo como objetivo padronizar códigos para caracteres alfa-numéricos. Com isso, computadores de diferentes fabricantes conseguiriam entender os códigos. Utiliza uma escala decimal de 0 a 127, onde esses numeros são convertidos para binários e o computador os processa, por exemplo, uma letra digitada por nós, humanos, corresponde a um desses códigos.

Diferentemente do ASCII, o Unicode utiliza códigos de valores mais altos do que 0 a 127. Com isso, é capaz de representar todos os caracteres específicos de diversos idiomas. Além disso, é capaz de definir uma correspondência entre simbolos e números. Na prática, um caracter Unicode utiliza 2 bytes o que acarreta que qualquer texto usa duas vezes mais espaço que no ASCII. Porém, o UTF-8 veio para salvar a pátria, reunindo a eficiência do ASCII e o âmbito do Unicode.
